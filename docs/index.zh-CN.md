# ScarGo 文档索引

该目录用于存放 ScarGo 四足机器人项目的工作文档。

## 语言约定

文档在具备条件时采用中英文双版本维护：

- 英文版：`name.md`
- 简体中文版：`name.zh-CN.md`

## 当前文档

### 项目概览

- `../README.md`：英文版项目概览
- `../README.zh-CN.md`：中文版项目概览

### 硬件

- `hardware-spec.md`：英文版机械与电气规格
- `hardware-spec.zh-CN.md`：中文版机械与电气规格

### 固件架构

- `architecture.md`：英文版固件架构说明
- `architecture.zh-CN.md`：中文版固件架构说明

## 文档规则

- 后续新增项目级文档时，默认同时提供英文版和简体中文版。
- 机械结构和电气接口发生变化时，应先更新文档，再固化代码中的假设。
- 通过标定和联调得到的新结论，应优先记录到硬件文档中。

## 备注

每当新增文档时，都应同步更新本索引。
