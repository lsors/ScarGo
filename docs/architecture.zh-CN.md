# ScarGo 架构说明

## 当前项目方向

ScarGo 是一个基于 ESP32-S3 的 12 自由度四足机器人 `ESP-IDF` 固件项目。

当前硬件方向如下：

- 以 ESP32-S3 作为唯一主控制器
- 使用 PCA9685 输出 12 路舵机控制信号
- 使用 MPU6050 获取 IMU 数据
- 使用 SSD1306 作为本地显示界面
- 通过 UART 接入 ELRS 遥控接收机
- 每条腿采用 tandem thigh/calf 传动结构

当前机械与电气定义请参考 `docs/hardware-spec.md` 和 `docs/hardware-spec.zh-CN.md`。

## 规划中的固件分层

- board 层：引脚映射、时钟、启动、板级基础服务
- driver 层：I2C、UART、PWM、PCA9685、MPU6050、SSD1306、按键、蜂鸣器、风扇
- device 层：舵机抽象、遥控输入抽象、IMU 抽象
- model 层：机身几何、腿部几何、校准表、关节映射
- control 层：运动学、姿态控制、步态生成、平衡控制、PID
- app 层：工作模式、界面、遥测、调试、故障处理

## 当前工程优先事项

- 先把硬件定义固化到文档里
- 为 ESP32-S3 建立清晰的板级引脚映射
- 按既定总线分配打通 PCA9685、OLED、MPU6050
- 定义舵机零位偏置与运动限位
- 进一步明确 tandem 小腿机构的关节映射
- 建立最小可用的站立与校准流程
