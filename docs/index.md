# ScarGo Documentation Index

This directory stores the working documentation for the ScarGo quadruped project.

## Language Convention

Documents are maintained in bilingual form when available:

- English: `name.md`
- Simplified Chinese: `name.zh-CN.md`

## Current Documents

### Project Overview

- `../README.md`: English project overview
- `../README.zh-CN.md`: Chinese project overview

### Hardware

- `hardware-spec.md`: English mechanical and electrical specification
- `hardware-spec.zh-CN.md`: Chinese mechanical and electrical specification

### Firmware Architecture

- `architecture.md`: English firmware architecture notes
- `architecture.zh-CN.md`: Chinese firmware architecture notes

## Documentation Rules

- New project-level documents should include both English and Simplified Chinese versions.
- Mechanical and electrical changes should be updated in documentation before code assumptions are locked in.
- Calibration-driven discoveries should be recorded in the hardware documents first.

## Notes

This index should be updated whenever new documentation is added.
